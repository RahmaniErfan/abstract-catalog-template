# Abstract Catalog
The Source of Truth for the Abstract Folder Ecosystem.

![Build Status](https://img.shields.io/github/actions/workflow/status/RahmaniErfan/abstract-catalog-template/validate.yml?branch=main)
![Total Libraries](https://img.shields.io/badge/libraries-1-blue)

This repository hosts the central index used by the [Abstract Folder](https://github.com/RahmaniErfan/abstract-folder) Obsidian plugin to discover, search, and install community libraries.

## Disclaimer
While we manually vet all submissions to this catalog, **third-party libraries are installed at your own risk.** Always review the content of a library before trusting it with your private data.

## For Authors
Want to see your library here? Check out our [Contributing Guide](CONTRIBUTING.md) to get started.

## Technical Details
The `catalog.json` file is designed for O(1) lookups and provides a global blacklist to protect users from malicious content instantly.
